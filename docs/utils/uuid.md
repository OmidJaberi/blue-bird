# UUID

The UUID module provides utilities for generating and converting UUID v4 identifiers.

---

# Features

- Generate random UUID v4 values
- Convert binary UUIDs to string form
- Generate UUID v4 values directly as strings

---

# UUID Format

UUID strings use the standard 36-character format:

```text
xxxxxxxx-xxxx-xxxx-xxxx-xxxxxxxxxxxx
```

Use `BB_UUID_BUF_LEN` when allocating buffers for UUID strings, since it includes the null terminator.

---

# Generating Binary UUIDs

Generate a random UUID v4 in binary 16-byte form:

```c
#include <blue-bird/utils/uuid.h>

uint8_t uuid[16];

if (bb_uuid_v4(uuid) != 0) {
    /* handle error */
}
```

---

# Converting UUIDs To Strings

Convert a binary UUID into its canonical string representation:

```c
#include <blue-bird/utils/uuid.h>

uint8_t uuid[16];
bb_uuid_t str;

bb_uuid_v4(uuid);

if (bb_uuid_to_string(uuid, str) != 0) {
    /* handle error */
}
```

---

# Generating UUID Strings Directly

Generate a UUID v4 directly into a string buffer:

```c
#include <blue-bird/utils/uuid.h>

bb_uuid_t str;

if (bb_uuid_v4_string(str) != 0) {
    /* handle error */
}
```

---

# Constants

| Constant | Description |
|---|---|
| `BB_UUID_STR_LEN` | Length of a UUID string without the null terminator |
| `BB_UUID_BUF_LEN` | Buffer size required for a UUID string including the null terminator |

---

# Notes

- UUIDs generated by this module are RFC 4122 version 4 UUIDs.
- Binary UUIDs are represented as 16-byte arrays.
- String UUIDs are always null-terminated.
